# Minimal Boilerplate for HTML / CSS / JS 

### Adapted from:
- Twitter bootstrap 
- Html5 Boilerplate

### Requires:
- sass
- fabric
- git

### Comes standard with:
- jQuery
- modernizr
- selectivizr

### Useful notes:
- compile css: ```fab css.compile```
- watch css: ```fab css.watch```
- deploy to staging: ```fab deploy.staging``` (requires setup)
- deploy to production: ```fab deploy.production``` (requires setup)
